How to use Regular Expressions?

Posted on 2019-08-01 |

Regular Expressions is a designed string that can be used in Java, JavaScript, Python and other language, It helps us to find different pattern in given string text. Regular Expressions first appeared in Books Introduction to Metamathematics written by Stephen Cole Kleene.

Regular Expressions always be supposed to a complex grammar, and I also learned it more than three times but still could not remember the detail usage. After thinking it over, it’s a wrong method to learn by searching but summarizing. Therefore, I summarize some Regular Expressions usages with JavaScript in this blog.

Validate——use as a RegExp object

  1. Validate email.

  2. Validate American phone number .

Replace——use as a parameter

  1. If you have a date in YYYY-MM-DD format and want to change into YYYYMMDD format, so you can use Regular Expressions in replace() method as a parameter.

  2. Delete all special characters in a string.

Slice——use its inner parameter

  1. If you have a date in YYYYMMDD format and want to change into YYYY-MM-DD format, you can also use replace there and the difference is using () to extract parameters.
