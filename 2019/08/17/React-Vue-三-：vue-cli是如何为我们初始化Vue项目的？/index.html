React&Vue(三)：vue-cli是如何为我们初始化Vue项目的？

Posted on 2019-08-17 |

相信对于每一个（像我这样的）前端新人来说，使用vue-cli、create-react-app这样成熟的脚手架，大大降低了搭建项目的门槛。这就和如果不会自己写作业，就直接拿别人写好的作业交上去是一个道理（我这是什么奇怪的比喻啊喂）。

许多人都认为前端入门的门槛很低，确实是的，太多的脚手架、UI框架、甚至界面框架，你都可以在github上找到，新手想写出一个自己的界面，一个星期完全足够了。在我看来，想要做一个合格的前端工程师，前端架构如何搭建、如何进行打包、如何区分开发、测试、产品环境，这些都是我们需要了解的更深层次的知识。

我以前一直觉得这些配置性的代码难以阅读难以理解，就一直逃避去面对他们。最近借着工作不多（外企真的养老啊……），看了webpack和vue-cli的文档，就顺便结合一下文档，自己分析一下Vue-cli的配置项和基本原理。(前两周把vue-cli升级到3.0了之后发现它把大部分配置项都隐藏了，难受，直接卸载）

不说废话，直接开始。
